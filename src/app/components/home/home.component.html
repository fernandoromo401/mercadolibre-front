Detector de Mutantes

Sistema que detecta si la secuencia de ADN pertenece a un Humano o a un Mutante

El front genera un ADN de forma aleatoria con las letras "A" "C" "G" "T" y siempre es una matriz cuadrada

Se lo envia al back para que analice si se trata de un Mutante o solo un Humano

Una vez detectado lo almacena en la base de datos para poder sacar un promedio mas adelante
